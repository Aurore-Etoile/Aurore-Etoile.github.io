html笔记

Author: Aurore  Date: December 12, 2019  22:31:30   Category: 前端笔记
  1. <!DOCTYPE html>
  2. <html lang="en">
  3. <html></html>：告诉浏览器这是html
  4. <head></head>：告诉浏览器关于微博页面的信息，该标签及其内容称为首部
    1. <title></title>：（一定要有）出现在浏览器窗口顶部
    2. <style type="text/css"></style>：告诉浏览器使用什么类型的样式
    3. <meta charset="utf-8">：在charset属性中指定字符编码
  1. <body></body>：页面主题包含所有内容和结构
    1. <a href="...#（id）" title="..."></a>：创建超文本链接
    2. <img src="..." ale="...">：html5中必须有alt
    3. <p></p>（块元素）    <q></q>    <blockquote></blockquote>
    4. <br></br>
    5. <em></em>    <strong></strong>
    6. <code></code>
    7. <pre></pre>
  1. http://www.facebook.com/index.html（用来获取资源的协议：//域中一个特定服务器的名字.域名/绝对路径）
    1. www.facebook.com：网站名
    2. facebook.com：域名（www是域中一个特定服务器的名字）
    3. <link type="text/css" rel="stylesheet" href="xxx.css">
    4. link：链入外部信息
    5. rel：指定了html文件和所连接文件之间的关系，此处为样式表
  1. （元素）.类名
  2. #id
  3. text
    1. text-align：对齐块元素中的所有内联元素
    2. text-decoration
    3. letter-spacing：字母之间设间距
    4. line-height：行间距
    5. color
    6. background-color
    7. background-image
    8. background-position
    9. background-repeat
    10. font-family：字体
    11. font-weight：文本粗细
    12. font-size
    13. border：边框
    14. border-style
    15. list-style：列表外观
    16. padding：内边距
  1. @font-face：建立一个字体，指定一个font-family名

@font-face {

font-family: "xxx";

src: url("xxx.woff");

url("xxx.ttf");

}

  1. @import
  2. @media
